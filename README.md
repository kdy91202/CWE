# CWE(Calculator Without E)
처음으로 안드로이드 계산기를 만들어 봤습니다. 버전 1.0
--------------------------------------------
지수 표기(E)가 없는 계산기로 소수점 연산은 되지 않습니다.

문제점 및 개선해야될점 점 :

1.(문제점)사칙 연산 순서가 제대로 작동을 안함

2.(문제점)제곱 연산 불가

3.(개선점)버튼 색깔 추가
